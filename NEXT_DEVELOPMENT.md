## 📝 각 기능 상세 설명

### 1. 서비스 소개 페이지

**라우터 추가**:
```javascript
{ path: '/about', component: About }
```

**주요 섹션**:
- 서비스 소개
- 사용 방법
- 주요 기능
- 기술 스택
- 팀 소개 (선택)

---

### 2. 문의/Q&A 페이지

**라우터 추가**:
```javascript
{ path: '/contact', component: Contact }
```

**주요 섹션**:
- 자주 묻는 질문 (FAQ)
- 문의하기 폼
- 연락처 정보

---

### 3. 소셜 공유 기능

**필요한 라이브러리**:
- 카카오톡 공유: Kakao SDK
- 페이스북 공유: Facebook SDK (선택)
- 링크 복사: Clipboard API

**구현 위치**: `AnalysisResult.vue`

---

### 4. 평가/피드백 기능

**데이터 구조**:
```javascript
{
  url: "https://...",
  rating: 5, // 1-5점
  feedback: "정확한 분석이었습니다",
  timestamp: "2025-01-15T10:30:00Z"
}
```

**저장 위치**: 로컬 스토리지 또는 별도 서비스
